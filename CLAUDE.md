# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This repository is for a submission to the **中国大学生计算机设计大赛 (China University Computer Design Competition)**, 2026 edition. The competition covers software R&D categories: 软件应用与开发, 物联网应用, 大数据应用, and 人工智能应用.

Competition instructions and templates are in `instruction&&requirments/`. The `.md_cache/` directory contains pre-converted markdown versions of the .docx/.xlsx files (auto-generated by a hook).

## Environment

- Python 3.12 with a `.venv` virtual environment
- Use `uv` for package management (never pip directly to base env)
- Activate venv: `source .venv/bin/activate`
- Installed packages: openpyxl, python-docx, lxml

## Competition Constraints

- Allowed AI tools are restricted to a specific list of 10 Chinese AI platforms (通义, 文心, DeepSeek, 智谱, Kimi, etc.) — see `instruction&&requirments/instruction/附件一：竞赛内容说明及分类.docx` for the full list
- Any AI-generated content must be disclosed in the submission form with tools and technical approach noted
- Team size: 2–9 authors from the same university, all undergraduates
- Submissions must be original work created between 2025.07–2026.05
- Videos must not exceed 10 minutes; file folders must contain no more than 10 files (use archives)
- Maps must comply with PRC regulations (审图号 required)
