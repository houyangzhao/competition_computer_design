# 校赛常见问题（FAQ）

> 赛项链接：https://jsjds.blcu.edu.cn/
> 东大创研竞赛链接：https://mp.weixin.qq.com/s/KigdeGKlZaMStcytZA_PIg
> 赛事平台链接：https://mse.wsetac.qq.com/u/aRd52kCTxVBh45hATCzM5Kw

| 编号 | 问题 | 回答 |
|------|------|------|
| 1 | 人工智能实践赛（三维建模与虚拟现实、自然语言处理、图像处理与模式识别方法研究、机器学习方法研究）这些做软件的话，除了只做软件的演示视频比较单薄，有关于演示视频更详细的要求吗？比如从设计开始讲述？ | 在符合竞赛规则的提前下，内容可由参赛团队自主决定。若只做功能演示内容会显得单薄，可加一些设计思想、技术创新等内容，大约四分之一作宣讲，剩余作展示和演示，合理安排队伍与指导老师的时间。现场答辩演示视频要思路清晰及相关内容。 |
| 2 | 一个人只能报名参加一个大赛赛道一次？ | 一个已报名参赛学生，在大赛11个大类赛道中，每个大类赛道每人最多只能报名1件作品，不可在同一个大类中报名多个作品，但可在不同大类赛道中分别报1件作品。 |
| 3 | 一个赛道报名之后能修改题目么？ | 在校赛作品提交截止日期之前，可以报名同一赛道大类下更换选题，需要确保更换的选题在通知的范围内。若已认真提交了最终的作品提交，在赛道层上不可换选题，若作品描述有变动则需要重新通过。需要证参赛作品与校赛上推荐的作品的核心思想一致，不可以大幅度更改作品的核心方向。 |
| 4 | 校赛报名之后还能增加或更换队伍成员吗？ | 在校赛作品提交截止时间之前可以增加或更换队伍成员。答辩后指名需满足一所院校全日制本科生，队伍人数2-9人，指导教师不超过2名。参赛者提交截止后填上报名后，若已校赛提交截止时间，原则上不允许增减或更换成员。在进入省赛阶段后，国赛上报名名单要确保与校赛的省赛名单排序一致，不可进入人员交换。 |
| 5 | 需要提交中的附件3（作品报告）是什么？在网盘中只有附件1与2，是不是对应类的作品开发与设计文档以及设计报告？ | 校赛要求提交的附件3（作品报告），已包含参赛类别提交的文件要求，只是不同通道会有差异。需要设置对应类别的文件要求：作品报告/作品设计书与开发文档。目前提前做好2025/2026参考材料，请大家关注大赛官网对应赛道通知。赛网站及省赛公告中有赛道模板提供，使用赛项校赛模板提交材料即可。 |
| 6 | 校赛只有软件研发类的大类吗？ | 软件学院只负责软件研发大类的校赛承办。其他大类由美术艺术学院等负责，在这边没有通知，请合同学了解东大双创微信公众号。 |

| 8 | 校赛只有软件研发类的大类吗？ | 软件学院只负责软件研发大类的校赛承办。其他大类由美术艺术学院等负责，请关注东大双创微信公众号。 |
| 9 | 校赛需要展示实物吗？晋级省赛后有多长时间可以进一步打磨作品？ | 根据赛事要求来进行，硬件是否展示由团队自行决定。 |
| 10 | 省赛会分为线上审核和线下答辩吗？省赛晋级国赛的比例大概是多少？ | 没有。 |
| 11 | 校赛要提交源码以及对应的安装包吗？Web开发必须要有域名吗？ | 源码和配套安装包属于建议提交项，并非强制要求，但为了让评审完整核验作品真实性与完整性，推荐提交；Web开发类作品不强制要求购买域名、上线公网，只需在本地完成环境搭建、功能实现，通过演示视频完整展示运行效果即可。 |
| 12 | 校赛是只需要交作品汇总表、作品报告和答辩PPT吗？答辩视频等是等省赛或国赛再交吗？ | 对，答辩视频后续根据省赛和国赛要求进行提交。 |
| 13 | 不需要在官网报名是吗？作品编号怎么写？ | 先不管网报名，作品编号先空着。 |
| 14 | 物联网赛道一定要有硬件实物吗？用于采集输入信号的硬件不太适合展示和购买怎么办？ | 根据赛事要求来进行，硬件是否展示由团队自行决定。 |
| 15 | 有没有用户手册和答辩PPT的要求或者模板？ | 没有。 |
| 16 | 请问是向邮箱提交了作品就算报名校赛了吗？ | 是，后续会有科协同学进行电话二次确认。 |
| 17 | 必须有指导教师吗？其他大类的校赛还没有发通知吗？ | 其他类比赛由艺术学院负责。链接：https://mp.weixin.qq.com/s/dbR25ICTVYRMNPATCjMFXw |
| 18 | 校赛需要提交作品演示视频吗？ | 非必需。 |
| 19 | 校赛报名需要在对应的网站上报名吗？还是说提交邮箱去答辩就可以了？ | 校赛阶段不用在官网报名，提交作品到邮箱就可以参加答辩了。 |
| 20 | 必须有指导老师吗？ | 不用。 |
| 21 | 作品信息概要表中作品编号填什么？ | 暂时不填。 |
| 22 | 截止时间是4.8中午12:00还是晚上12:00？ | 4.8中午12:00截止作品提交。 |
| 23 | 多次提交能以最后一次为准吗？ | 可以。 |
